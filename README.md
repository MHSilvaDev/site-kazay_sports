# 🏆 Kazay Sports - E-commerce Premium de Camisas Esportivas

Bem-vindo ao repositório da **Kazay Sports**, uma landing page de e-commerce moderna, responsiva e altamente profissional, desenvolvida para proporcionar uma experiência de compra premium no nicho de artigos esportivos.

![Hero Section Preview](images/preview.png) *(Placeholder para imagem de preview)*

## 🚀 Tecnologias Utilizadas

Este projeto foi construído com as tecnologias mais modernas do ecossistema front-end para garantir performance, escalabilidade e facilidade de manutenção:

- **React 19**: Biblioteca principal para construção de interfaces modulares e reativas.
- **Vite 8**: Ferramenta de build e dev server ultra-rápida.
- **Tailwind CSS v4**: Nova versão do framework de CSS, utilizando o plugin oficial `@tailwindcss/vite` para um sistema de design baseado em variáveis de CSS nativas.
- **Lucide React**: Biblioteca de ícones minimalistas e modernos.
- **Google Fonts**: Integração das fontes *Outfit* e *Inter*.

## 🎨 Design System & Estética

O site foi projetado com um visual "Dark Mode" elegante, inspirado em gigantes como Nike e Adidas, com um contraste vibrante em Verde Neon.

- **Paleta de Cores**:
  - `Base`: `#0B0B0B` (Preto Profundo)
  - `Primary`: `#00FF88` (Verde Neon/Esportivo)
  - `Secondary`: `#1A1A1A` (Cinza para áreas de destaque)
- **Tipografia**:
  - **Outfit**: Aplicada em todos os títulos (H1-H6) para um visual forte, agressivo e esportivo.
  - **Inter**: Aplicada em textos de corpo, botões e preços para garantir máxima legibilidade e conforto visual.

## ✨ Funcionalidades e Seções

1.  **Header Inteligente**: Sticky header com efeito *blur* (glassmorphism), busca integrada, contador de carrinho e menu mobile responsivo.
2.  **Hero Section**: Banner de alto impacto com tipografia estilizada, gradientes dinâmicos e botões de chamada para ação (CTA).
3.  **Diferenciais (Features)**: Destaque para benefícios como Frete Grátis e Qualidade Premium com ícones interativos.
4.  **Categorias**: Grid moderno com imagens de alta qualidade, transições suaves de zoom e badges informativos.
5.  **Grid de Produtos**: Listagem profissional de produtos com suporte a badges de "Novidade", "Descontos", avaliações e botões de compra rápida.
6.  **Prova Social (Testimonials)**: Seção de depoimentos de clientes reais utilizando cartões com efeito de vidro.
7.  **Footer Institucional**: Newsletter integrada, links rápidos de navegação, ícones de redes sociais e selos de formas de pagamento.
8.  **Favicon Customizado**: Ícone da aba do navegador personalizado com o logo "K" da marca.

## 🛠️ Como Executar o Projeto

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

1.  **Clone o repositório**:
    ```bash
    git clone https://github.com/MHSilvaDev/site-kazay_sports.git
    ```

2.  **Entre na pasta do projeto**:
    ```bash
    cd site-kazay_sports
    ```

3.  **Instale as dependências**:
    ```bash
    npm install
    ```

4.  **Inicie o servidor de desenvolvimento**:
    ```bash
    npm run dev
    ```

5.  **Acesse no seu navegador**:
    O Vite indicará a porta (geralmente `http://localhost:5173`).
